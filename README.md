## Installation (custom repository)

1. Open the **Dalamud Settings** menu in the game, either by typing `/xlsettings` in chat or by using the button at the bottom of the **Plugin Installer**.
2. Go to the **Experimental** tab.
3. In the **Custom Plugin Repositories** section, paste `https://raw.githubusercontent.com/richtertarnished/ukr_chat/master/repo.json` into the empty box at the bottom.
4. Click the **+** button.
5. Click the **💾** (save) button.

The plugin will then appear in the Plugin Installer, which you can open by typing `/xlplugins` in chat.
